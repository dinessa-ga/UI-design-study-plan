# Study Plan for UI/UX Design

This repository is a guide for learning UI/UX design. It includes projects, guides, and steps to create good user experiences. It helps beginners build skills in design thinking and tools like Figma.

## Files in This Repository

### prototype-phase.md
This file explains Phase 4: Prototyping. It focuses on a To-Do List app project. It covers different levels of prototypes, from paper sketches to digital designs in Figma. It also talks about interactions, animations, and testing with users.

### path-3-projects-ui.md
This file gives three project ideas for your UX/UI portfolio. The options are:
- An E-Learning platform for education.
- A marketplace for professional services.
- A health and wellness app.
It recommends starting with the E-Learning project to learn complex UI and UX.

### marketplace-services-app.md
This is a full project called "OficiosConnect." It is a marketplace for tradespeople and clients. It includes research, user flows, wireframes, design system, and prototypes. It shows how to design for both clients and professionals.

### ux_ui_guide.md
This is a complete guide for the UX/UI design process. It covers basics of UX and UI, user research, information architecture, user flows, and more. It also includes tips for Figma and design thinking.
